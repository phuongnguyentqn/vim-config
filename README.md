# nvim

## 1 - Dependencies

+ NeoVim: https://github.com/neovim/neovim
+ Vim Plug: https://github.com/junegunn/vim-plug

## 2 - Settings
Clone this repo to `~/.config/nvim/`
```
~/.config/nvim/init.vim
```

## 3 - Install plugins
+ Open nvim:
```
cd ~/.config/nvim/
nvim .
```
+ Install plugins in nvim
```
:PlugInstall
```
